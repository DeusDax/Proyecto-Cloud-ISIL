# Proyecto Cloud ISIL

Proyecto **Tactical Support Cloud** para la gestión de solicitudes de soporte técnico de Tactical IT S.A.C.

El código de la aplicación se encuentra en [`tactical-support-cloud`](./tactical-support-cloud). Consulta su [documentación](./tactical-support-cloud/README.md) para conocer la instalación, estructura, seguridad y despliegue.

## Inicio rápido

```bash
cd tactical-support-cloud
npm install
npm start
```

Después abre `http://localhost:3000`.
